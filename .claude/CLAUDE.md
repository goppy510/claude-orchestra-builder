# Claude Orchestra Builder

Interactive scaffolder for Claude Code orchestration environments.

## Language policy
- Internal documents (CLAUDE.md, rules, skills, agents, docs): English
- User-facing responses and conversations: Japanese (敬語)
- Code comments: English
- When generating files for target projects, follow this same policy

## Principles

- Generated files must follow official best practices
